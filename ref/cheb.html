cheb(n=0, kind=1, symbol='x')
Calculate Chebyshev polynomial of order "n" of first or second kind
using iterative formula. Return numpy.poly1d object.
 
 
Example
-------
 
>>> import cheb
>>> print cheb.cheb(2)
   2
2 x - 1
>>> print cheb.cheb(n=3, kind=2, symbol='y')
   3
8 y - 4 y
